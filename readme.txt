Connect your WordPress site with n8n workflow automation.

== Description ==

FlowBridge (n8n Edition) connects your WordPress site with n8n workflow automation. Configure webhooks to send data from posts, taxonomies, users, and contact forms to your n8n workflows.

== Installation ==

1. Upload the `flowbridge-for-n8n` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to FlowBridge Settings to configure your n8n webhook URL.
